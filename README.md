## Análisis de documentación

Se trabajará con RapidAPI, un centro de APIs del mundo, donde más de tres millones de desarrolladores encuentran, prueban y se conectan a miles de APIs, todo con una sola cuenta y clave de API.
Documentación: https://docs.rapidapi.com/ 

Se empleará la API de Aliexpress por ser un e-commerce, por lo cual podemos relacionarla con el retail Falabella. Para poder hacer uso de esta API, primero debemos suscribirnos en la API, podemos elegir un plan de gratuito para las pruebas.

- API: https://rapidapi.com/ecommdatahub/api/aliexpress-datahub
- Endpoint: https://aliexpress-datahub.p.rapidapi.com
- Métodos HTTP: GET y POST
- Parámetros: filter = baseRegion, key1 = value, key2 = value
- Código de estado HTTP: 200 OK
